# Nardi System - Backend

Attendance & Student Management Platform - Backend Models & Utilities

## Structure

- `models/` - Database models (Student, Attendance, Course, Batch)
- `utils/` - Utilities (DB connection, Auth, QR, Email)
- `types/` - TypeScript type definitions

## Usage

This backend code is used by the Next.js API routes in the frontend application.

The backend folder contains:
- MongoDB models and database operations
- Authentication utilities
- QR code generation
- Email sending functionality
- Type definitions

See main README.md for full documentation.
